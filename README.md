# Access Control BCB CF 🔐

Sistema de Controle de Acesso em conformidade com LGPD.

## ⚠️ Avisos de Segurança

### LGPD - Lei Geral de Proteção de Dados

Este sistema processa dados pessoais (nome, documento). Você é responsável por:

1. **Consentimento dos Visitantes** ✅
   - Obter consentimento antes de coletar dados
   - Informar sobre coleta e armazenamento

2. **Retenção de Dados** 📅
   - Dados são automaticamente deletados após 6 meses
   - Essa política está implementada no código

3. **Acesso Restrito** 🔐
   - Apenas pessoal autorizado deve acessar
   - Use senhas fortes
   - Mude senha padrão imediatamente

4. **Backup Seguro** 💾
   - Faça backup regularmente
   - Armazene em local seguro
   - Criptografe backups sensíveis
