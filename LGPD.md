# Política de Dados - LGPD

## 1. Coleta de Dados

Este sistema coleta:
- Nome do visitante
- Documento de identidade
- Empresa/Motivo da visita
- Horário de entrada/saída
- Nome do operador que registrou

## 2. Base Legal

A coleta é justificada por:
- Segurança da propriedade
- Cumprimento de obrigações legais
- Consentimento do visitante

## 3. Retenção

- Dados mantidos por **6 meses**
- Deletados automaticamente após este período
- Backup: máximo 1 ano (encriptado)

## 4. Direitos do Titular

O visitante tem direito a:
- Acessar seus dados
- Corrigir informações incorretas
- Solicitar exclusão
- Portabilidade de dados

## 5. Segurança

- Sistema de autenticação
- Acesso restrito
- Sem compartilhamento com terceiros
- Criptografia (quando em servidor)
